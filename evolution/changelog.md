# Changelog

Log of all updates to the Content Skill Graph.

When patterns change, hooks get updated, or new learnings emerge, log them here with date and rationale.

---

## Log Format

```
## YYYY-MM-DD

### [CATEGORY] Change Title

**What changed:**
Description of the change

**Why:**
Rationale based on performance data or strategic decision

**Files updated:**
- [[file1]]
- [[file2]]
```

---

## Categories

| Category | Description |
|----------|-------------|
| HOOK | New hook pattern, hook retirement, hook refinement |
| PLATFORM | Platform-specific rule change |
| VOICE | Brand voice adjustment |
| FORMAT | Content type specification change |
| SCHEDULE | Posting frequency or timing change |
| AUDIENCE | Audience segment update |
| WINNER | New winning pattern documented |
| LOSER | New losing pattern documented |
| SYSTEM | Skill graph structure change |
