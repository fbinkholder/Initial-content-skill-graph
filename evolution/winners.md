# Winners

Patterns that consistently outperform. Updated by Performance Analyst agent based on metrics.

When a pattern hits performance thresholds across multiple pieces of content, document it here for replication.

---

```

**First documented:** YYYY-MM-DD
**Status:** Testing / Confirmed / Evergreen

Description of what works

**Example:**

**Performance:**

- Metric 2: Y%

**Why it works:**
Analysis of why this resonates

**Replication notes:**
How to use this pattern again
```

A pattern qualifies as a "winner" when it exceeds these benchmarks:

| Platform | Metric | Threshold |
|----------|--------|-----------|

| TikTok | Engagement rate | >3% |
| Instagram | Engagement rate | >5% |

| YouTube | Avg view duration | >50% |

| X | Engagement rate | >2% |

## Graduation Path

```
Testing (48-72 hrs) → Confirmed (7 days) → Evergreen (30+ days)
```

**Testing:** One piece of content exceeded thresholds. Worth watching.

**Confirmed:** Pattern replicated across 3+ pieces with consistent performance.

**Evergreen:** Pattern still works after 30+ days. Add to core playbook.
