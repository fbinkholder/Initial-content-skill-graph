# Content Types

Format specifications for every content type Chromara produces.

---

## Short-Form Video

**Platforms:** TikTok, Instagram Reels, YouTube Shorts, Facebook Reels

| Attribute | Specification |
|-----------|---------------|
| Length | 45-90 seconds (sweet spot: 60 sec) |
| Aspect ratio | 9:16 vertical |
| Hook window | 1-2 seconds maximum |
| Captions | Always on (muted viewing default) |
| Text overlays | Required for key points |

**Structure:**
```
[0-2 sec] HOOK — grab immediately, text overlay
[2-45 sec] CONTENT — deliver value, show don't tell
[45-60 sec] CTA — soft close, waitlist if natural
```

**Production notes:**
- Write TikTok version first, adapt to other platforms
- Reels can be slightly more polished than TikTok
- YouTube Shorts extracted from best segments of long-form OR adapted from TikTok
- Facebook Reels adapted from Instagram Reels

## Instagram Carousel

**Platform:** Instagram (can adapt to LinkedIn)

| Attribute | Specification |
|-----------|---------------|
| Slide count | 7-10 slides |
| Aspect ratio | 1:1 or 4:5 |
| Slide 1 text | Max 7-10 words (hook) |
| Body slides text | Max 30 words per slide |
| Final slide | CTA with @chromarabeauty |

**Structure:**
```
Slide 1: Bold hook (7-10 words max)
Slides 2-8: One idea per slide (max 30 words each)
Final slide: CTA with handle
```

**Design requirements:**
- Brand colors: Violet #8C52FF, Near-Black #0D0D0F, Chrome Silver #E5E7EB
- Typography: Montserrat bold for headlines
- Readable at thumbnail size
- Consistent style across all slides
- No slide should require zooming to read

**Caption structure:**
```
[Paragraph 1] Hook, teach, or reframe. Deliver value immediately

[Paragraph 2] Deepen the point, add context, end with benefit-led CTA

Join the waitlist through the link in bio.

#hashtag1 #hashtag2 #hashtag3 (3-5 max)
```

## Long-Form Video

**Platform:** YouTube

| Attribute | Specification |
|-----------|---------------|
| Length | 8-15 minutes |
| Aspect ratio | 16:9 horizontal |
| Hook window | First 5 seconds |
| Chapters | Required in description |
| Thumbnail | Custom, readable text, high contrast |

**Structure:**
```
[0:00-0:30] HOOK — why this matters, what they'll learn
[0:30-2:00] CONTEXT — the problem, why it exists
[2:00-10:00] MAIN CONTENT — chapters for each section
[10:00-12:00] RECAP + CTA — summary, waitlist
```

**Title formulas:**
- "Why [X]% of People [Problem] (The Real Reason)"
- "I [Worked/Spent] [Time] [Doing X]. Here's What I Learned"
- "What Nobody Tells You About [Topic]"
- "[Number] Things [Audience] Gets Wrong About [Topic]"

**Production notes:**
- SEO-optimized title is critical (YouTube is a search engine)
- B-roll and visual variety required (no pure talking head)
- Chapter timestamps in description
- End screens for subscribe/waitlist

## Single Tweet

**Platform:** X

| Attribute | Specification |
|-----------|---------------|
| Length | Under 280 characters |
| Format | Lowercase |
| Hashtags | None |
| Links | In reply, not main tweet |

**Formulas:**
- Observation: "ever notice how [insight]"
- Stat punch: "[number]. [implication]"
- Sharp critique: "[industry behavior] and [contradiction]"
- Question: "why are we still [accepting broken thing]"

**Example:**
```
67% can't find their shade. 40% of foundation gets returned. $2B+ in waste yearly. and we keep launching more 40-shade ranges
```

## Twitter/X Thread

**Platform:** X

| Attribute | Specification |
|-----------|---------------|
| Length | 3-10 tweets |
| Format | Lowercase |
| First tweet | Hook, ends with 🧵 or "a thread:" |
| Each tweet | Standalone idea |
| Final tweet | Soft CTA |

**Structure:**
```
Tweet 1: HOOK — sharpest version, signals thread
Tweets 2-8: CONTENT — one idea per tweet, each works alone
Tweet 9-10: RECAP + CTA
```

**Rules:**
- Each tweet should work independently if seen alone
- Vary rhythm across tweets (don't repeat same structure)
- No hashtags
- Link in final tweet reply, not main thread

## Newsletter Email

**Platform:** Brevo (list #21)

| Attribute | Specification |
|-----------|---------------|
| Length | 800-1,500 words |
| Format | Plain text or minimal design |
| CTAs | One only |
| Sender | hello@chromarabeauty.com |
| Subject line | Lowercase, under 50 chars, personal |

**Structure:**
```
Subject line: Lowercase, personal, intriguing
Preview text: Extends the hook

Body:
- Personal opening (1-2 paragraphs)
- Main content (education, story, insight)
- Behind-the-scenes or exclusive element
- Single CTA

Sign-off: Faith
```

**Subject line examples:**
- "we got rejected"
- "something I've been thinking about"
- "the photo that reminded me why we started this"
- "a question for you"
- "this week was hard"

## Facebook Post

**Platform:** Facebook

| Attribute | Specification |
|-----------|---------------|
| Length | 150-400 words |
| Format | Conversational |
| Links | In comments, not caption |
| End with | Discussion question |

**Structure:**
```
[Opening] Personal story or relatable observation
[Middle] Context or insight
[End] Discussion question that invites participation
```

**Example:**
```
Question for you: when was the last time you bought a foundation and it actually matched on the first try?

Not "close enough." Actually matched.

I'm genuinely curious because in all my years on the Sephora floor, I could count those moments on one hand.

Drop your experience below 👇
```

## Pinterest Pin

**Platform:** Pinterest

| Attribute | Specification |
|-----------|---------------|
| Aspect ratio | 2:3 vertical (1000x1500px ideal) |
| Text overlay | Mandatory |
| Description | 150-300 characters (up to 500) |
| SEO | Keywords naturally woven in |

**Pin types:**
- Infographic (educational, problem/solution)
- Product visual (NovaMirror renders)
- Quote/stat pin (sharp insight as visual)
- How-to pin (shade matching education)

**Description structure:**
```
[Hook/stat] [Context/insight] [CTA with keywords]
```

**Example description:**
```
67% of people can't find a foundation that matches [Revieve 2024]. The problem isn't your undertone. It's how shades are manufactured. NovaMirror creates 4.5 million shade combinations from 9 refillable pods. Precision matching that finally works. Join the waitlist.
```

## Content Type Selection Guide

| Goal | Best Format |
|------|-------------|
| Discovery/reach | TikTok, YouTube Shorts |
| Education depth | YouTube long-form, Instagram carousel |
| Community building | Facebook post, Newsletter |
| Press/media visibility | X thread, X single tweet |
| Evergreen traffic | Pinterest, YouTube |
| Intimate connection | Newsletter |

---

## Quality Checklist (All Formats)

Before publishing any content:

- [ ] Hook lands in first 1-2 seconds/lines?
- [ ] One clear idea, one clear takeaway?
- [ ] Platform-native format (not just resized)?
- [ ] Brand voice consistent?
- [ ] No banned patterns (em dashes, antithetical parallelism, etc.)?
- [ ] Stats verified and within ceiling?
- [ ] CTA is benefit-led, not salesy?
- [ ] Would pass the "human wrote this" test?
